about

Masters By Research student at IIT Bombay, working under the guidance of Prof. Manas Thakur . My research interests lie in the areas of compilers and programming languages (PL) and my current work involves optimizations for Just-In-Time compilers within managed runtimes. In the coming years, I aim to explore advanced compiler optimizations, language design principles, and the integration of new paradigms into existing languages

projects

Mini-Interpreter

Built a miniature language runtime in Java with a custom lexer, parser, AST, symbol table, and interpreter. The project explores core compiler and virtual machine concepts through the end-to-end implementation of a language execution pipeline.

Java Language-design Virtual Machines
Dockerish

A Docker-like container runtime built from scratch using Linux namespaces, virtual networking, and OverlayFS. Supports image creation, layered filesystems, container lifecycle management, inter-container communication, and host-to-container port forwarding through a custom Dockerfile-inspired interface.

Linux Containers OverlayFS
Distributed-KeyValue

Developed a multi-tier key–value store in C featuring a thread-pooled TCP server, in-memory caching layer, and MySQL-backed persistence. Built a configurable load generator and conducted bottleneck analysis to evaluate throughput, latency, and resource utilization under CPU- and I/O-intensive workloads.

Distributed C Multithreading
